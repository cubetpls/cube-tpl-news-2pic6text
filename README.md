# 标准资讯 cube 模板，2*n 图 +6 文字资讯

## 介绍
标准资讯类模板，适合新闻资讯类场景。

## 使用
```
cubetool init <cubeid> -t https://github.com/ntt360/cube-tpl-news-2pic6text.git
```
或简写：
```
cubetool init <cubeid> -t ntt360/cube-tpl-news-2pic6text
```
